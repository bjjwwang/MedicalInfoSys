# 医案通 - 疑难杂症病例知识交易平台

## 项目愿景

打破疑难杂症治疗经验的信息孤岛，构建一个医疗案例知识交易平台，让成功治愈的经验可以被发现、被验证、被复用。

## 核心价值主张

- **对医生**: 将临床经验转化为可持续收益，获得同行认可
- **对患者**: 找到与自身相似的成功治愈案例，获取治疗思路参考
- **对药企/科研**: 获取真实世界数据 (RWD)，加速研发决策
- **对行业**: 推动医疗知识标准化流通，减少重复误诊

## 项目定位

**医学知识交易平台**（非在线问诊/诊疗平台）

平台提供的是治疗经验的参考信息，不直接提供诊疗服务，所有案例仅供参考。

## 目标用户

| 角色 | 核心需求 | 优先级 |
|------|----------|--------|
| 临床医生 | 查阅疑难病例参考方案、分享经验获取收益 | P0 |
| 患者/家属 | 寻找相似成功案例、了解治疗方向 | P0 |
| 医疗机构 | 病例库建设、医生培训 | P1 |
| 药企 | 真实世界数据采购、上市后研究 | P2 |
| 科研机构 | 案例数据用于研究 | P2 |
| 保险公司 | 治疗方案合理性评估 | P3 |
