✅ 保留（核心闭环必须）

  • 用户注册登录（手机号+短信）

  • 管理后台：数据审核、用户管理、订单查看
  • 基础运营统计报表

❌ 砍掉（非必须/后续再补）

  • 贵阳大数据交易所API对接
  • 区块链节点部署 + 智能合约
  • 区块链溯源可视化
  • AI脱敏引擎（医疗BERT NER）
  • Elasticsearch向量/语义搜索
  • 支付宝支付（先只做微信）
  • 收益分红/自动分账引擎

🎯 怎么算"验证成功"

上线后 1-2 个月内，建议看以下几个核心指标决定下一步是升级到完整版还是调整方向。

维度 关键指标 🏆 强信号（升级V1.0） ⚠ 弱信号（需要调整）
供给 上传用户数 / 活跃上传率 ≥ 100 用户上传，周活 20%+ < 20 用户，活跃几乎为零
需求 询单数 / 付费转化 ≥ 20 笔真实付费交易 只有询价无成单
定价 成交单价分布 有数据支撑做定价模型 价格全压在最低档
客诉 纠纷率 / 数据质量投诉 < 5%，用户愿意复购 > 20%，质疑数据真实性
合规 监管/采购方要求 开始有机构询问交易所资质 完全是C端零散交易
关键判断： 如果 3 个月内同时出现"付费交易 ≥ 20 笔"+"机构采购询问资质"两个信号，就说明商业模式成立且到了必须上合规/区块链的时候 → 升级V1.0。
