技术实现（MVP 简化版）：
  • 数据库：notifications 表（user_id / category / title / body / link / is_read / created_at）
  • 后端：3 个 API —— GET /notifications、GET /notifications/unread_count、POST /notifications/{id}/read
  • 前端：铃铛组件每 30 秒轮询未读数；消息列表支持筛选、标记已读、点击跳转
  • 暂不做实时 WebSocket 推送、邮件/短信通知、APP 推送（这些在升级包里）
