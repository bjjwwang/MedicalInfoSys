# 模块六：管理后台

## 功能清单

### 6.1 仪表盘
- 核心数据概览：用户数、案例数、交易额、日活/月活
- 趋势图表：注册趋势、案例增长、收入曲线
- 实时监控：在线用户、正在进行的交易

### 6.2 用户管理
- 用户列表（搜索、筛选、导出）
- 用户详情查看
- 账号状态管理（激活/暂停/封禁）
- 医生认证审核
- 信用分手动调整（需审批）

### 6.3 内容管理
- 病例审核队列
  - 待审核列表（按提交时间排序）
  - AI预审结果展示
  - 审核操作：通过/驳回/要求修改
  - 批量审核
- 病例管理
  - 下架/恢复
  - 编辑（紧急修正脱敏问题）
  - 推荐/取消推荐
- 举报处理

### 6.4 交易管理
- 交易流水查看
- 退款审批
- 异常交易监控
- 提现审批
- 对账管理

### 6.5 运营工具
- 首页推荐位管理
- Banner管理
- 专题/合集管理
- 站内通知/公告发送
- 优惠券/促销活动管理

### 6.6 数据分析
- 用户分析：注册转化漏斗、留存率、用户画像
- 内容分析：病种分布、案例质量分布、热门搜索词
- 交易分析：GMV、客单价、付费转化率
- 导出报表

### 6.7 系统设置
- 角色权限管理（RBAC）
- 操作日志查看
- 系统参数配置（分成比例、价格范围等）
- 审核规则配置

## 权限角色

| 角色 | 权限 |
|------|------|
| 超级管理员 | 全部权限 |
| 运营主管 | 用户管理、内容管理、运营工具 |
| 内容审核员 | 病例审核、举报处理 |
| 财务 | 交易管理、提现审批、对账 |
| 数据分析师 | 数据分析（只读） |
| 客服 | 用户查询、纠纷处理 |

## 迭代计划

| 版本 | 功能 | 优先级 |
|------|------|--------|
| MVP | 病例审核、用户管理（基础）| P0 |
| V1.0 | 仪表盘、交易管理、医生认证审核 | P0 |
| V1.5 | 运营工具、数据分析基础版 | P1 |
| V2.0 | 完整数据分析、权限体系、高级运营工具 | P2 |
