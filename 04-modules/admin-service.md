# 模块六：管理后台

## 功能清单

### 6.1 仪表盘
- 核心数据概览：用户数、数据资产数、交易额、数据分红总额、日活/月活
- 趋势图表：注册趋势、数据资产增长、交易收入曲线、分红分配趋势
- 实时监控：在线用户、正在进行的交易、交易所订单状态
- 交易所对接状态：连接状态、待审核资产数、本周结算状态

### 6.2 交易所对接管理（交易所集成仪表盘）
- 交易所连接状态监控
- 数据资产提交审核管理（批量提交/撤回）
- 交易所审核结果同步与通知
- 确权状态跟踪（待确权/已确权/确权失败）
- 交易所数据资产类目映射管理
- 交易所API调用日志与异常告警

### 6.3 区块链监控
- 链上交易记录浏览
- 数据浏览记录上链状态监控
- 确权信息上链验证
- 区块链节点健康状态
- 智能合约执行日志
- 异常链上行为告警

### 6.4 用户管理
- 用户列表（搜索、筛选、导出）
- 用户详情查看（含区块链钱包信息、数据资产持有情况）
- 账号状态管理（激活/暂停/封禁）
- 认证审核（数据提供者/医生/机构）
- 信用分手动调整（需审批）

### 6.5 数据资产合规审核队列
- 待审核数据资产列表（按提交时间排序）
- AI预审结果展示（含脱敏检查、质量评分、合规检查报告）
- 审核操作：通过/驳回/要求修改
- 批量审核
- 交易所驳回资产的复审处理
- 脱敏问题紧急处理（下架+修正）

### 6.6 数据资产管理
- 数据资产列表管理
  - 下架/恢复
  - 编辑（紧急修正脱敏问题）
  - 推荐/取消推荐
- 数据资产质量分布统计
- 举报处理

### 6.7 交易管理
- 交易流水查看（平台内交易 + 交易所交易）
- 退款审批
- 异常交易监控
- 提现审批
- 对账管理（每日支付渠道对账 + 每周交易所对账）

### 6.8 每周结算管理
- 交易所每周结算发起与确认
- 结算差异处理（平台记录 vs 交易所记录对比）
- 数据分红每周计算与分配
- 结算报表生成与导出
- 历史结算记录查询

### 6.9 收益分配监管
- 收益分配总览（交易收益 + 分红收益）
- 分配规则配置（各角色分成比例）
- 异常分配告警（单笔异常高额、频繁退款后再分配等）
- 链上分配记录与平台记录一致性校验
- 数据分红追溯审计（消费行为 → 浏览记录 → 分红分配链路）

### 6.10 运营工具
- 首页推荐位管理
- Banner管理
- 专题/合集管理（按疾病、按数据类型等）
- 站内通知/公告发送
- 数据资产促销活动管理

### 6.11 数据分析
- 用户分析：注册转化漏斗、留存率、用户画像（按角色分类）
- 数据资产分析：数据类型分布、质量分布、热门搜索词、交易所上架率
- 交易分析：GMV、客单价、付费转化率、交易所交易占比
- 分红分析：分红总额、分红触发率、人均分红收入
- 导出报表

### 6.12 系统设置
- 角色权限管理（RBAC）
- 操作日志查看
- 系统参数配置（分成比例、分红比例、价格范围、结算周期等）
- 审核规则配置
- 交易所对接参数配置（API密钥、回调地址等）
- 区块链节点配置

## 权限角色

| 角色 | 权限 |
|------|------|
| 超级管理员 | 全部权限 |
| 运营主管 | 用户管理、数据资产管理、运营工具 |
| 合规审核员 | 数据资产合规审核、举报处理、脱敏复核 |
| 交易所对接管理员 | 交易所对接管理、确权管理、每周结算 |
| 财务 | 交易管理、提现审批、对账、收益分配监管 |
| 区块链管理员 | 区块链监控、链上记录审计 |
| 数据分析师 | 数据分析（只读） |
| 客服 | 用户查询、纠纷处理 |
