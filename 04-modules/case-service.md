# 模块二：病例服务（核心模块）

## 功能清单

### 2.1 病例录入
- **结构化模板录入**
  - 预设模板：按病种提供不同的结构化模板
  - 字段引导：每个字段有示例和说明
  - 草稿保存：支持分多次填写
- **AI 辅助录入**
  - 上传非结构化病历文档（PDF/图片）
  - AI 自动提取结构化信息，用户确认修正
  - 智能推荐 ICD 编码和标签
- **影像/报告上传**
  - 支持 CT、MRI、X光、病理报告、检验报告
  - 自动水印和脱敏处理

### 2.2 自动脱敏流水线

```
原始内容 → NER识别PII → 规则引擎补充 → 人工复核 → 脱敏完成
```

脱敏范围：
- 患者姓名 → 隐去
- 身份证号 → 隐去
- 手机号/地址 → 隐去
- 医院名称 → 可选保留/隐去
- 日期 → 模糊化（精确到月或相对时间）
- 影像中的文字信息 → OCR检测 + 遮蔽

### 2.3 内容审核

审核维度：
1. **脱敏完整性**: PII是否完全清除
2. **医学合理性**: 诊断和治疗方案是否合理
3. **内容真实性**: 是否有明显杜撰/抄袭迹象
4. **合规性**: 是否涉及违禁药品/非法疗法

审核流程：
```
AI自动审核 → 通过则发布 / 存疑则转人工 → 人工审核 → 通过/驳回/要求修改
```

### 2.4 病例展示

公开展示（免费）：
- 标题、疾病分类、患者基本信息（年龄段/性别）
- 主诉和症状摘要
- 治疗类型和疗效结果
- 评分和评价摘要

付费解锁内容：
- 完整诊断过程
- 详细治疗方案（用药/手术/中医细节）
- 影像资料和检查报告
- 随访记录

### 2.5 病例搜索

搜索维度：
- 关键词全文搜索
- 按疾病分类/ICD编码筛选
- 按症状组合筛选
- 按治疗类型筛选
- 按疗效结果筛选
- 按评分/价格/时间排序

高级搜索：
- AI 语义搜索（"60岁男性反复发热两周找不到原因"）
- 相似病例推荐（基于向量相似度）

### 2.6 病例质量分级

| 等级 | 标准 | 展示权益 |
|------|------|---------|
| S级 | 罕见病 + 治愈 + 完整随访 + 同行高分 | 首页推荐、专题收录 |
| A级 | 信息完整 + 疗效确切 + 有随访 | 分类推荐 |
| B级 | 基本完整，有参考价值 | 正常展示 |
| C级 | 信息不够完整，但有部分参考 | 低优先展示 |

## API 设计（核心接口）

```
POST   /api/v1/cases                   创建病例（草稿）
PUT    /api/v1/cases/{id}              更新病例
POST   /api/v1/cases/{id}/publish      提交审核/发布
GET    /api/v1/cases/{id}              获取病例详情（根据权限返回不同内容）
GET    /api/v1/cases/{id}/full         获取完整病例（需购买或作者本人）
DELETE /api/v1/cases/{id}              删除病例
GET    /api/v1/cases                   病例列表（支持筛选排序分页）
GET    /api/v1/cases/search            搜索病例
GET    /api/v1/cases/{id}/similar      获取相似病例
POST   /api/v1/cases/{id}/media        上传病例附件
POST   /api/v1/cases/ai-extract        AI提取结构化信息
GET    /api/v1/cases/me                我的病例列表
```
