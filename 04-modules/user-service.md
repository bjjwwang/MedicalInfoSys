# 模块一：用户服务

## 功能清单

### 1.1 注册/登录
- 手机号 + 验证码注册登录
- 微信/支付宝 OAuth 登录
- 小程序静默登录
- JWT Token 管理（access_token + refresh_token）

### 1.2 用户资料
- 基本信息管理（昵称、头像、简介）
- 角色选择（患者/医生/机构）
- 个人主页（发布的案例、收到的评价）

### 1.3 医生认证
- 认证流程：
  1. 填写真实姓名、医院、科室、职称
  2. 上传执业医师资格证照片
  3. 上传工作证/胸牌（辅助验证）
  4. 后台人工审核（对接国家医师注册查询系统校验）
  5. 审核通过，获得认证标识
- 认证等级：
  - 初级认证：基本信息 + 资格证
  - 高级认证：+ 医院官方确认函 / 学术成果

### 1.4 信用体系
- 信用分维度：
  - 案例质量（同行评审得分）
  - 交易信誉（纠纷率、退款率）
  - 活跃贡献（上传频率、讨论参与度）
- 信用分影响：
  - 高分：案例优先展示、更高提现额度
  - 低分：限制上传、降低曝光

### 1.5 钱包与提现
- 余额管理
- 充值（微信/支付宝）
- 提现（银行卡，T+1到账）
- 收支明细

## API 设计（核心接口）

```
POST   /api/v1/auth/sms-code          发送验证码
POST   /api/v1/auth/login              登录
POST   /api/v1/auth/refresh            刷新token
GET    /api/v1/users/me                获取当前用户信息
PUT    /api/v1/users/me                更新用户信息
POST   /api/v1/users/certification     提交医生认证
GET    /api/v1/users/{id}/profile      获取用户公开资料
GET    /api/v1/users/me/wallet         获取钱包信息
POST   /api/v1/users/me/withdraw       申请提现
```
